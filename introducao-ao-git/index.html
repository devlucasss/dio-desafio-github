Criando repositório para o Desafio da DIO

Reforçando meu conhecimento com git

Usa-se gitbash para:

  • navegar entre as pastas do computador
  • criar novas pastas/projetos/arquivos
  • criar sua chave SSH
  • inicializar o git nos projetos com o git init
  • dar clone em repositorios que estao no github
  • adicionar novas alteraçoes ao repositorio local
  • dar commit no repositorio local
  • empurrar para o repositorio remoto (Github)
